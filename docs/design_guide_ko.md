# 디자인 가이드: 다크모드 스타일 비교 및 명세

뽐뿌 모바일 미러링 프로젝트에서 구현된 두 가지 다크모드 스타일의 디자인 원칙과 기술적 명세를 정리한 문서입니다.

## 1. 스타일 비교 요약

| 항목 | 뽐뿌 순정 버전 (`m_ppom_dark.html`) | 네이버 스타일 버전 (`m_naver_dark.html`) |
| :--- | :--- | :--- |
| **디자인 컨셉** | 뽐뿌 고유의 다크모드 감성 유지 | 네이버 웨일/앱 기반의 프리미먼 감성 |
| **메인 배경색** | 어두운 그레이 / 블랙 계열 | 딥 차콜 (`#191a1c`) |
| **요소 배경색** | 다크 그레이 | 소프트 차콜 (`#222426`) |
| **강조 색상** | 블루 / 그레이 (기본값) | 네이버 그린 (`#03c75a`) |
| **기본 텍스트** | 연한 그레이 / 화이트 | 쿨 그레이 (`#e1e2e4`) |
| **보조 텍스트** | 흐린 그레이 | 뮤티드 블루 그레이 (`#b0b2b5`) |
| **경계선** | 기존 얇은 구분선 | 정갈한 슬레이트 구분선 (`#2e3033`) |

## 2. 네이버 스타일 컬러 팔레트 (Hex Codes)

- **Main Background**: `#191a1c` (시각적 피로도를 낮추는 네이버 특유의 배경색)
- **Card/Header Background**: `#222426` (콘텐츠 영역에 깊이감을 주는 색상)
- **Primary Text**: `#e1e2e4` (선명하면서도 눈이 편안한 텍스트색)
- **Naver Green**: `#03c75a` (활성 탭, 링크 호버, 강조 라인에 사용되는 브랜드 컬러)
- **Border/Divider**: `#2e3033` (미니멀한 영역 구분을 위한 색상)

---

## 3. 주요 요소별 스타일링 가이드

### 헤더 (Header)
- **로고 처리**: `filter: brightness(0) invert(1)` 속성을 사용하여 기존 블루톤의 뽐뿌 로고를 깔끔한 화이트로 반전시켰습니다.
- **높이 및 레이아웃**: 레이아웃이 깨지지 않도록 `54px` 높이를 고정하고 상단에 고정되도록 처리했습니다.

### 내비게이션 탭 (Navigation Tabs)
- **활성 상태**: 선택된 탭(`.on`) 하단에 `3px` 두께의 네이버 그린 라인을 추가하고 글자색도 그린으로 변경하여 직관성을 높였습니다.

### 리스트 및 링크 (Lists & Links)
- **텍스트 가독성**: 리스트 내 제목과 설명 텍스트를 일괄적으로 `#e1e2e4`로 고정하여 가독성을 확보했습니다.
- **호버 인터랙션**: 게시글 제목에 마우스를 올리면 텍스트가 네이버 그린으로 부드럽게 변하도록 설정했습니다.

---

## 4. 기술적 구현 특징

### 스타일 오버라이드 (CSS Injection)
뽐뿌의 기존 스타일시트(`mobile.css` 등)가 가진 강력한 우선순위를 극복하기 위해 다음과 같은 전략을 사용했습니다:
- **CSS 변수 활용**: `:root`에 테마 변수를 정의하여 일관성을 유지했습니다.
- **강제 적용**: 중요도가 높은 스타일에는 `!important`를 사용하여 기존 인라인 스타일이나 복잡한 선택자 규칙을 오버라이드했습니다.
- **Z-Index 최적화**: 헤더가 광고나 다른 유동 요소에 가려지지 않도록 `z-index: 9999`를 부여했습니다.
