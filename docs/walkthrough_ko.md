# 프로젝트 가이드: 뽐뿌 모바일 페이지 로컬라이징 및 다크모드 구현

이 문서는 뽐뿌 모바일 페이지의 자산을 로컬로 미러링하고, 다크모드(순정 및 네이버 스타일)를 구현한 과정과 결과에 대해 설명합니다.

## 1. 주요 성과

### 자산 로컬라이징 (Localization)
- **외부 자산 다운로드**: `CDN2`, `CDN3`, `static` 등 외부 서버에 흩어져 있는 이미지, JS, CSS 파일을 모두 `_external/` 폴더로 다운로드하여 로컬에서 완벽하게 동작하도록 했습니다.
- **재귀적 CSS 스캔**: CSS 파일 내부의 `url()` 참조(폰트, 배경 이미지)까지 추적하여 다운로드함으로써 레이아웃 깨짐을 방지했습니다.

### 다크모드 구현
- **순정 스타일 (`m_ppom_dark.html`)**: 뽐뿌 본연의 다크모드 디자인을 유지하며 인코딩 및 자산 경로만 해결했습니다.
- **네이버 스타일 (`m_naver_dark.html`)**: 네이버의 다크모드 감성을 이식하여 배경색(#191a1c), 강조색(네이버 그린 #03c75a), 화이트 로고 반전 등을 적용했습니다.

### 기술적 개선
- **인코딩 해결**: 기존 `EUC-KR` 페이지를 `UTF-8`로 일괄 변환하여 한글 깨짐 문제를 근본적으로 해결했습니다.
- **실행 스크립트 고도화**: `mirror_site.py`에 인자 전달 기능을 추가하고, 이미지 중복 다운로드 방지 로직 및 강제 업데이트(`-f`) 옵션을 도입했습니다.

## 2. 사용 방법

### 간편 실행 (`localize.bat`)
복잡한 파이썬 명령축 대신 배치 파일을 통해 간편하게 변환할 수 있습니다.
```cmd
localize.bat [원본파일.html] [결과파일.html]
```
*예시: `localize.bat m_dark.html m_ppom_dark.html`*

### 파이썬 스크립트 (`mirror_site.py`)
직접 파이썬으로 실행할 경우 다양한 옵션을 사용할 수 있습니다.
```bash
python mirror_site.py [input.html] [output.html] [-f]
```
- `-f`, `--force`: 이미 존재하는 이미지도 강제로 다시 다운로드합니다. (JS/CSS는 항상 업데이트됨)

### 2. 결과물 확인:
- 브라우저에서 `http://localhost:8000/` 뒤에 원하는 파일명을 입력하여 접속하세요.
- **핵심 파일 (루트)**: `m_naver_dark.html`, `m_ppom_dark.html`
- **기타 파일 (html/ 폴더)**: `html/m_skill.html`, `html/m_list_skill.html` 등
