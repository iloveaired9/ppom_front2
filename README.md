# 뽐뿌 모바일 로컬라이징 및 다크모드 프로젝트

이 프로젝트는 뽐뿌(Ppomppu) 모바일 웹 페이지의 자산을 로컬로 미러링하고, 한글 인코딩 문제를 해결하며, 사용자 맞춤형 다크모드 테마를 구현하는 도구 및 결과물 모음입니다.

## 🌟 주요 기능

- **완벽한 로컬 미러링**: 외부 CDN 자산(이미지, JS, CSS)을 로컬로 다운로드하고 경로를 자동 수정하여 오프라인에서도 동작하게 합니다.
- **인코딩 자동 변환**: 기존 EUC-KR 페이지를 UTF-8로 변환하여 브라우저에서의 한글 깨짐 현상을 근본적으로 해결합니다.
- **다크모드 커스텀 테마**:
  - **순정 스타일 (`m_ppom_dark.html`)**: 뽐뿌 고유의 다크모드 디자인 유지.
  - **네이버 스타일 (`m_naver_dark.html`)**: 네이버 다크모드 색감(#191A1C, Naver Green) 적용.
- **재귀적 자산 수집**: CSS 내부의 폰트 및 배경 이미지까지 추적하여 수집합니다.

## 📁 프로젝트 구조

- `m_naver_dark.html`: 네이버 스타일 다크모드 (핵심 결과물)
- `m_ppom_dark.html`: 뽐뿌 순정 다크모드 (핵심 결과물)
- `mirror_site.py`: 로컬라이징 핵심 파이썬 스크립트
- `localize.bat`: 윈도우용 간편 실행 배치 파일
- `html/`: 메인 페이지, 리스트 페이지 등 기타 로컬라이징 결과물
- `docs/`: 구현 과정 및 디자인 가이드 (국문 문서)
- `_external/`: 미러링된 외부 자산 폴더 (Git 관리 제외)

## 🚀 사용방법

### 페이지 로컬라이징 (변환)
1. **배치 파일 사용**:
   ```cmd
   localize.bat [원본.html] [결과물.html]
   ```
2. **파이썬 직접 실행**:
   ```bash
   python mirror_site.py [input.html] [output.html] [-f]
   ```
